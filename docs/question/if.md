# if

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
if (true) {
  function foo() {
    console.log('BFE')
  }
}
if (false) {
  function bar() {
    console.log('dev')
  }
}

foo()
bar()
```
