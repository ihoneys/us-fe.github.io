# array keys

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
console.log(Reflect.ownKeys([]))
console.log(Reflect.ownKeys([,]))
console.log(Reflect.ownKeys([1,,2]))
console.log(Reflect.ownKeys([...[1,,2]]))
```
