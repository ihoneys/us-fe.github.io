# RegExp.prototype.test

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
console.log(/^4\d\d$/.test('404'))
console.log(/^4\d\d$/.test(404))
console.log(/^4\d\d$/.test(['404']))
console.log(/^4\d\d$/.test([404]))
```
