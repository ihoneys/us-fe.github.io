# min max

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
console.log(Math.min())
console.log(Math.max())
console.log(Math.min(1))
console.log(Math.max(1,2))
console.log(Math.min([1,2,3]))
```
