# RegExp

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
const arr = ['a', 'b', 'c', '1']
const regExp = /^[a-z]$/gi
const chars = arr.filter(elem => regExp.test(elem))
console.log(chars)
```
