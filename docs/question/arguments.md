# arguments

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
function log(a,b,c,d) {
  console.log(a,b,c,d)
  arguments[0] = 'bfe'
  arguments[3] = 'dev'
 
  console.log(a,b,c,d)
}

log(1,2,3)
```
