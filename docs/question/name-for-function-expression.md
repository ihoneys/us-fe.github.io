# name for Function expression

## Question
What does the code snippet to the right output by `console.log`?

```js
function a(){
}
const b = function() {
  
}

const c = function d() {

  d = 'e'
  console.log(typeof d)
}

console.log(typeof a)
console.log(typeof b)
console.log(typeof c)
console.log(typeof d)
```
