# precedence

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
let a = 1
console.log(a +++ a)

let b = 1
console.log(b + + + b)

let c = 1
console.log(c --- c)

let d = 1
console.log(d - - - d)
```
