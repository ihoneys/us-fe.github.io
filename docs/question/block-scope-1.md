# block scope

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
for (var i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 0)
}

for (let i = 0; i < 5; i++) {
  setTimeout(() => console.log(i), 0)
}
```
