# Grid Layout 2

Layout the items in a grid so that:

*   items have minium width of 100px and fill up the space
*   place as many items in a row as possible
*   gap between items is 10px

```html
<div class="container">
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
</div>
```
