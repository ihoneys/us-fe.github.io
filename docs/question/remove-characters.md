# remove characters

Given a string contaning only `a`, `b` and `c`, remove all `b` and `ac`.

```ts
removeChars('ab') // 'a'
removeChars('abc') // ''
removeChars('cabbaabcca') // 'caa'
```

> What is the time and space complexity of your approach?
