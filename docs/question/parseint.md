# parseInt

## Question
What does the code snippet to the right output by `console.log`?

## Snippet
```js
console.log(['0'].map(parseInt))
console.log(['0','1'].map(parseInt))
console.log(['0','1','1'].map(parseInt))
console.log(['0','1','1','1'].map(parseInt))
```
